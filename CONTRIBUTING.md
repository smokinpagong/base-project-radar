# Contributing

   Thanks for your interest in Base Project Radar! This is a small learning project, but contributions are welcome.

   ## Ways to help
   - Suggest a project to add (open an Issue with the project name, category, and link)
   - Report a bug (open an Issue describing what went wrong)
   - Suggest a feature from the [ROADMAP](ROADMAP.md)

   ## Code style
   - Plain HTML, CSS, and JavaScript only — no frameworks or build tools
   - Keep everything in `index.html` unless a change clearly needs its own file
   - Match the existing naming style in `SEED_PROJECTS` when adding new projects
