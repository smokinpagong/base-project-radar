# 🛣️ Roadmap

## ✅ v0.1 — MVP (done)
- [x] Homepage with project grid
- [x] Search by name, summary, and tags
- [x] Category filters (DeFi, NFT, Infra, Social, Tooling)
- [x] Sort by newest / A–Z
- [x] Project detail popup (modal)
- [x] Submit-a-project form (saved locally in the browser)
- [x] Bookmarking (saved locally in the browser)
- [x] Starter seed data of real Base ecosystem projects

## 🔜 v0.2 — Real data sharing
- [ ] Move data from a hardcoded list into a real shared database, so submissions show up for *everyone*, not just the person who submitted them
- [ ] Add a basic review step before a submitted project appears publicly (to avoid spam)

## 🔜 v0.3 — Better browsing
- [ ] Project logos / images on cards
- [ ] Filter by tag, not just category
- [ ] "Trending" or "recently added" highlight section
- [ ] Pagination or infinite scroll once the list gets long

## 🔜 v0.4 — Community features
- [ ] Upvotes or simple reactions on projects
- [ ] Comments or short notes from the community
- [ ] An "About / how this list is curated" page

## 🌟 v1.0 — Public launch
- [ ] Custom domain
- [ ] Contribution guidelines for open-source submissions via GitHub
- [ ] Analytics to see which categories people care about most
