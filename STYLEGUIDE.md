# Style Guide

   A quick reference for the visual language of Base Project Radar.

   ## Colors
   - Background (void): `#0a0e14`
   - Primary accent (signal blue): `#0052ff`
   - Live/positive accent (scan green): `#39ff8a`
   - Card surface (panel): `#131a26`
   - Hairlines (grid line): `#232c3d`
   - Secondary text (fog): `#8893a6`
   - Primary text (paper): `#eaedf2`
   - Warning accent (amber): `#ffb454`

   ## Fonts
   - Headings: Space Grotesk
   - Body text: Inter
   - Tags, labels, code-like text: JetBrains Mono

   ## Status colors
   - Live → green dot
   - Beta → blue dot
   - Building → amber dot
